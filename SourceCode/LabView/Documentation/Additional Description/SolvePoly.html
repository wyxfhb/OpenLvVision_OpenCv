The function solvePoly finds the complex roots of a polynomial equation:

{{< katex >}}$ \sum_{i=0}^{n} \texttt{coeffs}[i] x^{n-i} = \texttt{coeffs}[0] x^n + \texttt{coeffs}[1] x^{n-1} + \dots + \texttt{coeffs}[n] = 0 ${{< /katex >}}

The function computes all complex roots of the polynomial and stores them in the roots array.
