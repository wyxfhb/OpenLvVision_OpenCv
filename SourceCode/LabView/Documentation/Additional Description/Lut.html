The function LUT fills the output array with values from the look-up table. Indices of the entries are taken from the input array. That is, the function processes each element of src as follows:

{{< katex >}}$ \texttt{dst}(I) \leftarrow \texttt{lut}(\texttt{src}(I) + d) ${{< /katex >}}

where

{{< katex >}}$ d = \begin{cases} 0 & \text{if } \texttt{src} \text{ has depth } \texttt{CV\_8U} \\ 128 & \text{if } \texttt{src} \text{ has depth } \texttt{CV\_8S} \end{cases} ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation
