The function cv::dft performs one of the following:

  • Forward Fourier transform of a 1D vector of N elements:
    {{< katex >}}$ Y = F^{(N)} \cdot X ${{< /katex >}}

    where

    {{< katex >}}$ F^{(N)}_{jk} = \exp\!\left(-\, 2\pi i\, \frac{jk}{N}\right), \qquad i = \sqrt{-1} ${{< /katex >}}
  • Forward 2D Fourier transform of an $M \times N$ matrix:
    {{< katex >}}$ Y = F^{(M)} \cdot X \cdot F^{(N)} ${{< /katex >}}

For detailed information, please refer to the OpenCV documentation
