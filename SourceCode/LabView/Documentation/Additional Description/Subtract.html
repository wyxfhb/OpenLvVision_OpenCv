The function subtract calculates:

  • Difference between two arrays (when both input arrays have the same size and the same number of channels):
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{src1}(I) - \texttt{src2}(I)) \quad \texttt{if mask}(I) \ne 0 ${{< /katex >}}
  • Difference between an array and a scalar (when src2 is constructed from Scalar or has the same number of elements as src1.channels()):
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{src1}(I) - \texttt{src2}) \quad \texttt{if mask}(I) \ne 0 ${{< /katex >}}
  • Difference between a scalar and an array (when src1 is constructed from Scalar or has the same number of elements as src2.channels()):
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{src1} - \texttt{src2}(I)) \quad \texttt{if mask}(I) \ne 0 ${{< /katex >}}
  • Reverse difference (SubRS variant) between a scalar and an array:
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{src2} - \texttt{src1}(I)) \quad \texttt{if mask}(I) \ne 0 ${{< /katex >}}

Where I is a multi-dimensional index of array elements. In case of multi-channel arrays, each channel is processed independently.

{{< alert context="warning" text="Saturation is not applied when the arrays have the depth CV_32S. You may even get a negative value in the case of overflow." />}}
