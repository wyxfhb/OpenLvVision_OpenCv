The function completeSymm copies the lower or upper half of a square matrix to its opposite half while keeping the diagonal unchanged:

{{< katex >}}$ \texttt{m}_{ij} = \texttt{m}_{ji} \quad \text{for } i > j \quad \text{if } \texttt{lowerToUpper} = \texttt{false} ${{< /katex >}}
{{< katex >}}$ \texttt{m}_{ij} = \texttt{m}_{ji} \quad \text{for } i < j \quad \text{if } \texttt{lowerToUpper} = \texttt{true} ${{< /katex >}}
