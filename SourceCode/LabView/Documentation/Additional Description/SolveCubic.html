The function solveCubic finds the real roots of a cubic equation:

  • If coeffs is a 4-element vector:
    {{< katex >}}$ \texttt{coeffs}[0] x^3 + \texttt{coeffs}[1] x^2 + \texttt{coeffs}[2] x + \texttt{coeffs}[3] = 0 ${{< /katex >}}
  • If coeffs is a 3-element vector (assumes monic polynomial):
    {{< katex >}}$ x^3 + \texttt{coeffs}[0] x^2 + \texttt{coeffs}[1] x + \texttt{coeffs}[2] = 0 ${{< /katex >}}

The roots are stored in the roots array.
