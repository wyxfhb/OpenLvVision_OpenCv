The function meanStdDev calculates the mean and the standard deviation of array elements independently for each channel and returns them via the output parameters:

{{< katex >}}$ \begin{array}{l} N = \sum\limits_{I, \ \texttt{mask}(I) \ne 0} 1 \\ \texttt{mean}_c = \frac{\sum\limits_{I : \ \texttt{mask}(I) \ne 0} \texttt{src}(I)_c}{N} \\ \texttt{stddev}_c = \sqrt{\frac{\sum\limits_{I : \ \texttt{mask}(I) \ne 0} \left( \texttt{src}(I)_c - \texttt{mean}_c \right)^2}{N}} \end{array} ${{< /katex >}}
