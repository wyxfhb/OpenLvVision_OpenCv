The function divide performs per-element division of two arrays or a scalar by an array:

  • Division of one array by another:
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{src1}(I) \cdot \texttt{scale} / \texttt{src2}(I)) ${{< /katex >}}
  • Division of a scalar by an array (when there is no src1):
    {{< katex >}}$ \texttt{dst}(I) = \texttt{saturate}(\texttt{scale} / \texttt{src2}(I)) ${{< /katex >}}

Different channels of multi-channel arrays are processed independently.

{{< alert context="warning" text="In case of floating point data there is no special defined behavior for zero src2(I) values. Regular floating-point division is used. Expect correct IEEE-754 behaviour for floating-point data (with NaN, Inf result values).Saturation is not applied when the output array has the depth CV_32S. You may even get result of an incorrect sign in the case of overflow." />}}
